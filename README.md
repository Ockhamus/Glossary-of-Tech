## Glossary-of-Tech

### 🧐 Overview
Glossary of Tech는 개발, IT 분야 등 다양한 기술 용어를 정리하기 위한 공간입니다.  
빠르게 변화하는 기술 세계에서 헷갈리거나 생소한 용어들을 우리만의 언어로 작성하고 이해하는 것을 목표로 합니다.  

### 🚀 Get Started
Glossary-of-Tech는 다음과 같은 방법으로 진행됩니다.  

1. **새 용어 제안**
    - 작성하고 싶은 용어가 있다면, Ockhamus의 **Glossary of Tech Project**에 Issue 등록
2. **브랜치 생성 및 파일 작성**
   - Git Branch를 이용하여 용어.md 파일 작성
3. **Pull Request (PR) 제출**
   - 작성 완료된 파일은 PR에서 리뷰
4. **리뷰 및 병합**
    - PR이 리뷰를 거쳐 main 브랜치에 병합 및 Issue 종료

</br>

**문제 발견 시:**
- 기존 용어 내용에 문제가 있거나 수정이 필요한 경우, **Issues**에 등록해 주세요.
